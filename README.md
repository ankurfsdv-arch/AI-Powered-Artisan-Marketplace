## 🚀 Key Features

### 🛠 Artisan Ecosystem
* **Smart Dashboard:** Real-time tracking of sales, orders, and customer engagement.
* **Product Management:** Advanced CRUD operations for artisan products with category tagging.
* **Stories Manager:** Unique section for artisans to share the heritage and process behind their crafts.

### 🤖 AI & Advanced Services
* **Translation Service:** Automatic multi-language support for global reach.
* **AI Product Discovery:** Intelligent search and trending product sliders based on user behavior.
* **Dynamic Pricing:** Tools for artisans to manage competitive pricing effectively.

### 🔐 Secure Infrastructure
* **Auth Flow:** Robust OTP-based mobile entry and multi-role (User/Artisan) signup.
* **Supabase Backend:** Real-time database updates and secure edge functions.

## 📦 Installation & Setup

1. **Clone the Project:**
   ```bash
   git clone [https://github.com/ankurfsdv-arch/AI-Powered-Artisan-Marketplace.git](https://github.com/ankurfsdv-arch/AI-Powered-Artisan-Marketplace.git)
   cd "AI-Powered-Artisan-Marketplace"

   Install Dependencies:
   npm install


Environment Configuration:

Create a .env file in the root and add:

Code snippet
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
Start the Development Server:
npm run dev
    ```
